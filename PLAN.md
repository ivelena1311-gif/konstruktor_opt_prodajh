## Этап 2 — Разработка MVP

Цель: рабочее приложение по публичной ссылке с демо-режимом.

### Wizard (4 шага)
- [ ] Шаг 1 — Изделие и фото: тип, название, upload до 6 фото, состав, плотность, размеры
- [ ] Шаг 2 — Цены и партия: опт/РРЦ/себест./MOQ/срок + автовычисление маржи
- [ ] Шаг 3 — Рынок: chips-выборы канала, сегмента, географии, сезона, поле ЦА
- [ ] Шаг 4 — Усиление: chips акцентов, боль байера, чекбоксы выхода
- [ ] Прогресс-бар, навигация «Назад/Далее/Пропустить», кнопка «Заполнить примером»
- [ ] Автосохранение данных в localStorage

### Генерация и экономика
- [ ] API-слой: сборка промпта, запрос к Claude API, парсинг и маппинг ответа
- [ ] Калькулятор экономики: маржа%, прибыль с партии, точка безубыточности
- [ ] Режим «Быстрый старт»: fallback при минимуме данных — результат всё равно выдаётся

### Экран результата
- [ ] Таб «Карточка» — предпросмотр итоговой карточки изделия
- [ ] Таб «Презентация» — PDF-презентация для байера
- [ ] Таб «Скрипты» — 3 скрипта с кнопками «Скопировать»
- [ ] Таб «Экономика» — расчёт маржи и прибыли с партии
- [ ] Экспорт PDF (html2pdf.js или Puppeteer)

### Инфраструктура
- [ ] Деплой на Vercel + env-переменные (Claude API key в Secrets)
- [ ] Демо-режим: 1 клик → готовый пример результата

- [ ] Провести онбординг: дать ссылку, наблюдать за первым сеансом

- [ ] Улучшить промпты, если качество текстов недостаточное

Цель: развитие продукта на основе данных и обратной связи.

- [ ] Блок «Вероятность продажи»: AI анализирует риск нереализации и рекомендует объём
- [ ] Кнопка «Усилить карточку» — перегенерация с другим акцентом
- [ ] AI-фотосессия: очистка фона + студийная обработка
- [ ] История проектов: личный кабинет, сохранение и повторная выгрузка PDF
- [ ] Режим «Профи»: полная PDF-презентация 10 слайдов с аналитикой рынка
- [ ] Интеграция с CRM (AmoCRM, Bitrix24)
- [ ] Управление подпиской в личном кабинете (/billing)
- [ ] Пакетный режим: несколько изделий в одном PDF (коллекция целиком)

## Контроль качества перед запуском

- [ ] Все unit-тесты зелёные
- [ ] E2E «Заполнить примером → скачать PDF» работает в Chrome и Safari
- [ ] Время от «Сгенерировать» до результата < 30 секунд
- [ ] PDF открывается и читается на мобильном и десктопе
- [ ] Платёж проходит в тестовом режиме
- [ ] Нет P0-багов: приложение не падает при пустых полях или ошибке Claude API
